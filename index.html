How to Implement Dark Mode Logo Switching

This guide explains how to switch logos between light and dark mode in a newsletter or website.

Step 1 – Choose Default Logo for Light Mode

Light mode uses a white or light background. So we use a dark logo:

images/new-logo-dark.png

Step 2 – Create Logo for Dark Mode

Dark mode uses a dark background. Dark logos become unreadable, so we use a light version:

images/new-logo-light.png

Step 3 – Add Dark Theme CSS

We toggle a class called dark on the body element: 

body.dark {
  background-color: #121212;
}

body.dark #dateText {
  color: #ffffff;
}

Step 4 – Modify the HTML

Add:

  • id="kwLogo" to the image
  • data-light and data-dark attributes
<img
  id="kwLogo"
  src="images/new-logo-dark.png"
  data-light="images/new-logo-dark.png"
  data-dark="images/new-logo-light.png"
/>

Step 5 – Add JavaScript

The JavaScript:

  • Detects dark mode
  • Toggles the class on body
  • Switches the logo source
function applyTheme(isDark) {
  document.body.classList.toggle("dark", isDark);
  logo.src = isDark ? logo.dataset.dark : logo.dataset.light;
}
